    module XPath2::NodeNavigator

    Overview

    NodeNavigator provides cursor model for navigating the XML data. Documents planning to provide XPath support should implement this module

    Defined in:

    xpath2.cr

    Instance Method Summary

    • #copy : NodeNavigator
